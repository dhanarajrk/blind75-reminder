## ✨ What is this?

Blind 75 Reminder is a minimal, distraction-free app designed to help you **actually stick to DSA practice**.

Instead of solving problems once and forgetting them, this app:
- reminds you when to revisit problems
- builds streaks
- tracks your learning consistency

---

## 💡 Why use it?

Because solving once is not enough.

This app helps you:
- 🔁 **Revise smartly** (spaced repetition)
- 🔥 **Build streaks**
- 📅 **Stay consistent**
- 🎯 **Focus on what matters**

No clutter. Just progress.

---

## 🌸 Features

- ✔️ Solve / Struggle tracking  
- 🔥 Streak calendar  
- ⏰ Smart reminders  
- 📊 Topic-wise progress  
- 🎨 Clean aesthetic UI (light + dark mode)
